## Hey welcome to the php course design of `LUNOVA` team members!

### What this course will cover?

This course will cover from the very begining of PHP to the advanced concepts of PHP. You will learn how to create dynamic web applications using PHP and MySQL.This course is designed for beginners who have little to no experience with PHP and also experienced PHP developer.
